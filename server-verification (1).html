Anforderungen der Server Verification

Du akzeptierst alle aktuellen und zukünftigen Anforderungen der Server Verification. Diese Regeln sind verbindlich und müssen vollständig eingehalten werden.

Serverban bei unangemessenen Inhalten

Du akzeptierst, dass dein Server dauerhaft gebannt wird, wenn du unangemessene, illegale oder regelwidrige Inhalte erstellst oder verbreitest.

Keine Bau-Rechte für Fremde

Du akzeptierst, dass du keinen fremden Personen („Randoms") Bau-Rechte oder administrative Berechtigungen auf deinem Server erteilst.

Professioneller Discord-Auftritt

Du akzeptierst, dass dein Discord-Server professionell gestaltet ist – mit klaren Regeln, einem einheitlichen Design und einer strukturierten Kanalübersicht.

Freundliches Teammitglied

Du akzeptierst, dass jedes Teammitglied deines Servers stets freundlich, respektvoll und hilfsbereit gegenüber der Community auftritt.

Aktiver Server

Du akzeptierst, dass dein Server regelmäßig aktiv ist und die Community kontinuierlich betreut wird. Inaktive Server verstoßen gegen diese Bedingungen.

Null-Fehler-Toleranz

Du akzeptierst, dass du alle Regeln fehlerfrei einhältst. Fehler, auch unbeabsichtigte, können zur Sanktionierung deines Servers führen.

Sofortiger Ban bei Verstößen

Du akzeptierst, dass selbst bei kleineren Regelverstößen dein Server ohne Vorwarnung direkt und dauerhaft gebannt werden kann.
